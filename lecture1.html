Lecture 1: Introduction to Web Development

In this lecture, we learn about the basic concepts and overview of Web Development.

🌐 Web Development - Networking Concepts

1️⃣ IP Address & Internet Protocol

IP Address: Unique number assigned to every computer connected to the Internet.
Example: 123.45.6.7

2️⃣ Client-Server Model

Computer A --HTTP Request--> Server (coderarmy.in)
Server --HTTP Response--> Computer A

3️⃣ Dynamic vs Static IP

Static IP → Permanent (e.g., 192.168.1.1)
Dynamic IP → Changes frequently (e.g., 123.12.1.5)

4️⃣ Common Ports

  • HTTP → Port 80
  • HTTPS → Port 443
  • FTP → Port 21
  • SMTP (Mail) → Port 25

5️⃣ IPv4 vs IPv6

IPv4 → 32-bit address (e.g., 192.168.1.1)
IPv6 → 128-bit address (e.g., 2001:0db8::1)

6️⃣ MAC Address

Unique hardware address for network devices.
Example: 00:1A:2B:3C:4D:5E

7️⃣ Private vs Public IP

Private IP → Used inside local network (e.g., 192.168.x.x)
Public IP → Globally unique address assigned by ISP

Ranges:
- Private: 10.0.0.0 – 10.255.255.255
- Public: 8.8.8.8 (Google DNS)

8️⃣ Router, Switch, Hub

[Computer A] --\
                [Switch] -- [Router] -- Internet
[Computer B] --/

9️⃣ Web Development Client-Server Flow

Client → Sends Request (HTML/CSS/JS)
Server → Processes → Returns Response (Data/HTML)

🔟 MERN Stack (Popular Web Development Stack)

Frontend → React.js
Backend → Node.js + Express.js
Database → MongoDB
