Lecture 2: What is HTML

This lecture covers what HTML is and its basic concepts.

📚 HTML - Hyper Text Markup Language

1️⃣ What is HTML?

HyperText Markup Language → Structure web content

HyperText:

Links connect pages → Web of information
Example:
<a href="page2.html">Go to Page 2</a>

Markup Language:

Tags mark content meaning:
<p>This is a paragraph.</p>
<h1>Main Title</h1>

6️⃣ Horizontal Rule & Line Break

<hr /> → Horizontal line
<br /> → Line break

Example:
221B Baker St.<br /> London, UK

7️⃣ Lists

<ol>
  <li>Boil water</li>
  <li>Add tea, milk</li>
</ol>
