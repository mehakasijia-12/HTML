Lecture 4: HTML Boilerplate Code | Div | Class | ID Notes

In this lecture, we explore HTML Boilerplate Code, div, class, and id in detail.

1️⃣ Boilerplate Code

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Content here
  </body>
</html>

2️⃣ File Paths

Relative Paths:

./dog.png
../cat.png
Beta/cow.png

Absolute Paths:

C:\Users\JohnDoe\Documents\report.docx
/Users/JohnDoe/Documents/report.docx

Example Usage:

<img src="./dog.png">

3️⃣ ASCII & UTF-8

A → 65 → 01000001
UTF-8 → Character ↔ Number
(1, 2, 3, or 4 bytes)

4️⃣ Div, Class & Id

<div>
  <h1>Heading</h1>
  <p>Paragraph</p>
</div>

Class Example:
<div class="box">...</div>

Id Example:
<div id="main">...</div>
