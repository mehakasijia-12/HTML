Lecture 6

Topics: Deploy Your First Project | Media Tag | Multi Page Website

In this lecture, we will learn about deploying a project, using media tags, and creating a multi-page website.

1️⃣ Why HTML Media?

Browsers used plugins (Flash, Silverlight) → Inefficient & Insecure.
Solution → HTML5 <video> & <audio> Tags.

2️⃣ Basic Video Embed

<video src="video.mp4" controls></video>

Common Attributes:

width="640" height="360"
autoplay muted controls
loop poster="thumb.jpg"

3️⃣ Audio Embed

<audio src="theme.mp3" controls></audio>

4️⃣ Multiple Sources for Compatibility

<video controls>
  <source src="video.webm" type="video/webm">
  <source src="video.mp4" type="video/mp4">
  Sorry, your browser doesn't support video.
</video>

5️⃣ Accessibility (Subtitles & Captions)

<video controls>
  <source src="vid.mp4">
  <track src="captions.vtt" kind="captions" srclang="en" label="English">
</video>

6️⃣ Comparison of Methods

| Feature      | <source> Method | src Attribute |
|------------- |-----------------------|-------------  |
| Compatibility| Excellent             | Good          |
| Simplicity   | Complex               | Simple        |
| Fallback     | Yes                   | No            |

7️⃣ Special Symbols (HTML Entities)

© → © or ©
® → ® or ®
™ → ™ or ™
< → <
> → >
& → &
